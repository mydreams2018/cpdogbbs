#### 安装教程

1. 下载此项目源代码.
2. 需要有nodejs环境
3. 进入项目根目录npm install 安装依赖
4. 需要配置你们的本地dns映射 www.kungreat.cn 为你们本地ip地址
5. npm start 启动项目
