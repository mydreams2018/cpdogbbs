# cpdogbbs

#### 介绍
基于react 架构的开源bbs论坛 ,所有功能已经完成。可以作为简易交流论坛使用。
完善了很多机制,包括贴子的审核,管理员的权限。
在线演示地址: https://www.kungreat.cn/

#### 安装教程

1. 下载此项目源代码.
2. 需要有nodejs环境
3. 进入项目根目录npm install 安装依赖
4. 修改.env 相关配置文件
5. npm start 启动项目/npm run build 打包项目
